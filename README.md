# Qur'an Application

A modern, accurate, and respectful Qur'an application built with Flutter, featuring:

- **Authentic Uthmani Arabic Text** - 100% accurate, unmodified
- **Dual Translations** - Bangla and English for every ayah
- **Audio Recitation** - High-quality tajweed with ayah-by-ayah playback
- **Modern UI/UX** - Minimal, distraction-free, spiritually calm interface
- **Offline Support** - Full access without internet connection
- **Search & Navigation** - Search by Surah, Ayah, or keywords
- **Personal Features** - Bookmarks, notes, and reading progress

## Features

### Core Reading
- Surah list with Arabic and English names
- Ayah-by-ayah reading view
- Toggle Bangla/English translations
- Word-by-word highlight
- Last-read ayah memory

### Audio Features
- High-quality Qur'an recitation
- Ayah-by-ayah playback
- Repeat ayah option
- Adjustable playback speed
- Offline audio download

### Personalization
- Bookmark ayahs
- Add personal notes (stored locally)
- Reading progress tracker
- Customizable font sizes

### Design
- Dark mode & night reading mode
- Soft Islamic color palette
- Elegant Arabic typography
- One-hand friendly navigation

## Getting Started

1. Install Flutter dependencies:
```bash
flutter pub get
```

2. Run the app:
```bash
flutter run
```
